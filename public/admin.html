STRIPE PAYMENT

Stripe processes card payments securely. Connect your live Stripe account to start accepting real payments.

Checking Stripe status...
  1. Go to dashboard.stripe.com/apikeys
  2. Make sure you are in Live mode (toggle top-left of Stripe dashboard)
  3. Copy your Secret key (starts with sk_live_)
  4. In Replit, open the Secrets panel and set STRIPE_SECRET_KEY to that value
  5. Restart the app — Stripe will be live immediately
LIVE PAYMENTS

Your store is ready for real payments. Use your sk_live_ key — customers will be charged real money. Make sure your Stripe account is fully verified before going live.

CHECKOUT FALLBACK

If Stripe is not connected, the store automatically uses a direct checkout that creates orders without requiring payment. Use this for testing or manual/invoice-based orders.

ALWAYS ON

Direct checkout is always available as a fallback. When Stripe is connected, it's used first. If Stripe fails or isn't set up, orders still go through.
